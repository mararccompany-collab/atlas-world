# 🌍 أطلس العالم - Atlas World

موسوعة تفاعلية شاملة لجميع دول العالم بالعربية والإنجليزية

## ✨ المميزات

- 📍 **أكثر من 100 دولة** مع تفاصيل شاملة
- 🔍 **بحث ذكي** يدعم الحروف الإنجليزية والعربية
- ⚔️ **معلومات عسكرية** دقيقة لكل دولة
- 💰 **بيانات اقتصادية** وتفاصيل البنوك الرئيسية
- 👑 **معلومات سياسية** عن قادة الدول والوزراء
- 🗺️ **الموقع الجغرافي** والأنهار والبحيرات
- 🏛️ **المعالم السياحية والتاريخية**
- 📰 **روابط الأخبار المباشرة**

## 🚀 البدء السريع

### المتطلبات
- Node.js 16+
- npm أو yarn

### التثبيت

```bash
# استنساخ المشروع
git clone https://github.com/YourUsername/atlas-world.git
cd atlas-world

# تثبيت المكتبات
npm install

# تشغيل خادم التطوير
npm run dev
```

ثم افتح المتصفح على: `http://localhost:5173`

### البناء للإنتاج

```bash
npm run build
```

## 🛠️ التكنولوجيا المستخدمة

- **React 19** - مكتبة بناء الواجهات
- **TypeScript** - للأمان والموثوقية
- **Tailwind CSS** - تصاميم حديثة وسريعة
- **Vite** - بناء سريع وفعال
